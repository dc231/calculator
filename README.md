# Calculator
A clean and responsive calculator built with HTML, CSS, and Vanilla JavaScript. It performs essential arithmetic operations with a modern interface and smooth user experience.

# Live Demo
https://dc231.github.io/calculator/

# Tech Stack
- HTML
- CSS (Grid Layout, Responsive Design)
- JavaScript (Vanilla)

# Features
- Perform basic operations: addition, subtraction, multiplication, division
- Percentage handling (`%` as `/100`)
- Clear (`AC`) and delete (`DEL`) functionality
- Supports decimals and double zero input
- Prevents invalid operator usage
